homework--volume-of-sphere-assignment by cooljeanius

This was my first homework assignment for my Intro to C class.

Unfortunately my professor and teaching assistant don't accept homework assignments via GitHub yet (we have to use BlackBoard), but just in case they ever do...

Dependencies

• Standard ANSI C '99 Libraries
• GCC
• Some sort of text editor
Optional:
• Xcode, if you want to be able to open the Xcode project
• CodeBlocks, if you want to be able to open the CodeBlocks project

Install

$ cd /path/to/wherever/you/placed/volumeofsphere
$ gcc -o somethingotherthanvolumeofsphere ./volumeofsphere.c
$ ./somethingotherthanvolumeofsphere

Feel free to move the output to somewhere in your $PATH after that.
